üéÆ Interactive Activities

1. Kahoot Game (10 minutes)

We'll kick things off with an exciting Kahoot quiz featuring 8-10 questions on general programming concepts. This is your chance to warm up, have fun, and compete with your peers!

üèÜ Prizes: The top 3 scorers will receive swag prizes and points will be added to the CodeCore leaderboard!

2. Variable Types Visual Summary (5 minutes)

Watch as we create a visual "cheat sheet" using large Post-it notes. Each variable type will be displayed with clear examples that you can reference throughout the session.

3. Live Array Demonstration - The Main Event!

This is where things get really interesting! We'll use physical chairs and volunteers (including faculty members) to represent array elements in memory.

How It Works:

  • Five chairs labeled with indices (0-4) represent an array
  • Volunteers hold cards showing index numbers, memory addresses, and values
  • Teams compete on whiteboards to answer challenges
  • One card says "ERROR" to demonstrate out-of-bounds access
  • Earn candy and points for correct answers!

Example challenges you'll solve:

  • "What's stored at index 2?"
  • "Change the value at index 4 to 'Dr. Torres'"
  • "What happens when you access index 6?" (spoiler: ERROR!)

4. Loops in Action

Building on the array setup, we'll demonstrate how loops work by "visiting" each chair (array element) in sequence. You'll see exactly how code like for (int i = 0; i < array.length; i++) translates to real actions.

5. Conditional Logic Challenges

Work with your team to predict outputs from code snippets and fill in missing pieces. Perfect for reinforcing your understanding of if/else statements!

6. Slido Q&A Session

Have questions? Submit them anonymously through Slido throughout the workshop. Faculty members will address all questions at the end of the session.
